# 使用
1、 打开Poedit。  
2、 打开PoeditHelper。  
3、 载入po文件。  
4、 选择翻译类型。  
5、 点击Poedit中待翻译文本，PoeditHelper会自动进行翻译。  
5、 点击PoeditHelper中翻译后的文本，会自动填入Poedit。
