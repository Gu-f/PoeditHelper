# 使用
1、 Open Poedit.  
2、 Open PoeditHelper.  
3、 Load po-file.  
4、 Select translation type.  
5、 Click on the text in Poedit and PoeditHelper will translate it automatically.  
6、 Clicking on the translated text in PoeditHelper automatically fills in Poedit.
